Project Webot

Due to Covid-19 and lockdown rules.  Team Farmbot had the task of testing Webot on home machines. 

 

Webots is an open source and multi-platform desktop application used to simulate robots. It provides a complete development environment to model, program and simulate robots.

It has been designed for a professional use, and it is widely used in industry, education and research. Cyberbotics Ltd. maintains Webots as its main product continuously since 1998.


As a team we were tasked to download webots with our home machines and document our findings.

I tested 3 computer setup scenerios and recorded my experience. 


PC (Windows 10)

PC (running VirtualBox with Ubuntu 18.04 LTS)

Mac (High Sierra)



PC Virtual Machine Mac

PC Specs & Test

  • Windows 10 (64-bit)  i5 3.2GHz, 1.5 TB HDD
  • RAM - 22 GB
  • Graphics Card(s) - ATI Radeon HD 4550, Intel HD Graphics 4600.   Note: Webots officially supports only recent NVIDIA and AMD graphics adapters.

 

After installing webots, I went straight into getting started with webots.  The tutorials are straight forward and makes for easy navigation around the program itself.  I got as far as the first tutorial (hands-on #3) with creating your first simulation in webots.  Unfortunately it would not display what should be a rectangle arena.  I was only presented with a white screen on all my monitors with 'not responding' message which  did not change with time.  What a shame!


VirtualBox VM Specs & Test

  • Ubuntu (64-bit), 125 TB HDD
  • RAM - 8 GB
  • Graphics Card(s) - VMSVGA  Note: Webots officially supports only recent NVIDIA and AMD graphics adapters.

Installing Virtual Box was very straight forward in creating a virtual machine.  As suggested by webots, Ubuntu 18.04 LTS was the recommended distribution to install to meet the minimum requirements for running the application.

Again I followed the tutorials and same as the pc setup  I got as far as the first tutorial (hands-on #3) with creating your first simulation in webots. Unfortunately again it would not display what should be a rectangle arena. I was only presented with a blue screen but I could still navigate around the program but nothing would show if I added extra things to the simulation.
